# Changelog

All notable changes to Auto Filer will be documented in this file.

## [0.0.1] - 2026-04-03

### Added
- Initial release of Auto Filer
- Interactive file creation mode with guided prompts
- JSON configuration file support
- Support for creating nested directories automatically
- Global and local installation options
- `create` command for interactive file creation
- `load` command for configuration file-based file creation
- `init` command to generate sample configuration files
